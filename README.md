# Grocery E-Commerce MERN Stack Project

This repository will host a production-ready MERN stack grocery e-commerce application.

## Project Structure

- `backend/` - Node.js, Express.js, MongoDB, Mongoose, authentication, payments, and APIs
- `frontend/` - React, Vite, Tailwind CSS, routing, state management, and UI

## Phase 1 Goal

Create the project foundation, folder structure, Git setup, and environment configuration before building features module by module.

## Next Step

Backend scaffold and Express server setup.
